# Biscuit & Batter

Biscuit & Batter is a beginner-friendly dessert blog website built with HTML and CSS.  
The site includes a homepage, an about page, and a contact page with a styled contact form.

## Features

- Responsive homepage with recipe cards
- About Me page
- Contact page with form fields
- Navigation bar linking all pages
- Styled hero banner images
- Footer with social media links
- Mobile-friendly layout using CSS media queries
- Footer layout

## Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries

2. Open the project folder.
3. Open homepage.html in your browser.

You can also open aboutme.html and contactpage.html directly in your browser.
